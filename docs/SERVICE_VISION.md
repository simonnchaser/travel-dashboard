# 여행 대시보드 서비스 비전

## 🎯 핵심 가치: "자유로운 여행 설계"

### 서비스 한 줄 요약

**개인 맞춤형 여행 계획부터 실시간 일정 관리, 그리고 경험 공유까지 - 여행의 전 과정을 지원하는 올인원 플랫폼**

---

## 📊 시장 배경 및 기회

### 현재 여행 트렌드 변화

- **패키지 여행 감소** → 개인 자유 여행 증가
- **전체 일정 패키지** → 도시별/구역별 일일 투어 선호 (마이리얼트립 등)
- **획일화된 여행** → 개인화된 맞춤 여행 수요 급증

### 시장의 갭 (Gap)

기존 패키지 여행이 제공하던 서비스 중 **현재 플랫폼들이 못 채우는 것들:**

1. ✅ **전체 일정 관리** - 마이리얼트립은 개별 투어만 판매
2. ✅ **이동 경로 최적화** - 구글맵/네이버지도는 단순 경로만 제공
3. ✅ **숙박 연계 관리** - 예약 사이트는 예약만 담당
4. ✅ **여행 경비 관리** - 따로 엑셀이나 가계부 앱 사용
5. ✅ **실전 경험 기반 정보** - 블로그는 글 중심, 전체 일정 파악 어려움

→ **우리 서비스가 이 모든 갭을 통합적으로 해결**

## 🚀 서비스 차별점

### 1. 기존 서비스와의 비교

| 기능        | 구글맵/네이버지도 | 여행 블로그  | 마이리얼트립 | **우리 서비스**    |
| ----------- | ----------------- | ------------ | ------------ | ------------------ |
| 경로 안내   | ⭕                | ❌           | ❌           | ⭕                 |
| 일정 관리   | ❌                | 제한적       | 개별 투어만  | ⭕ 통합 관리       |
| 비용 관리   | ❌                | 제한적       | ❌           | ⭕                 |
| 실시간 수정 | ❌                | ❌           | ❌           | ⭕                 |
| 경험 공유   | ❌                | ⭕ (글 위주) | ❌           | ⭕ (구조화된 일정) |
| 개인화      | ❌                | ❌           | ❌           | ⭕                 |

### 2. 핵심 차별점: "퍼즐 맞추기식 일정 설계"

**기존 방식의 문제:**

- 블로그: 글로만 작성 → 전체 일정 파악 어려움, 지도상 위치 확인 불편
- 엑셀/노션: 수동 관리 → 번거롭고, 지도 연동 없음
- 구글맵: 저장된 장소만 표시 → 시간대별 일정 관리 불가

**우리의 해결책:**

- ✅ 지도 + 타임라인 통합 뷰
- ✅ 드래그 앤 드롭으로 일정 재배치
- ✅ 경로 자동 계산 (대중교통/도보/차량)
- ✅ 비용 실시간 집계
- ✅ 예약 링크/파일 통합 관리

## 🎪 서비스 활용 시나리오

### 실제 사용 사례: 동유럽 가족 여행

**배경:** 부모님 모시고 프라하 → 빈 → 부다페스트 자유여행 진행 중

**여행 전 (계획 단계):**

1. 각 도시별로 가고 싶은 장소 추가
2. 지도에서 위치 확인하며 동선 최적화
3. 대중교통 경로와 소요시간 확인
4. 숙소, 투어 예약 정보 한곳에 정리
5. 일자별 예산 계획

**여행 중 (실행 단계):**

1. 실시간으로 일정 수정 (날씨, 컨디션에 따라)
2. 즉흥적으로 발견한 장소 즉시 추가
3. 퍼즐 맞추듯이 일정 사이에 끼워넣기
4. 지출 내역 바로바로 기록
5. 교통 경로 확인하며 이동

**여행 후 (공유 단계):**

1. 완성된 일정이 자동으로 구조화됨
2. 사진/메모 추가하여 스토리텔링
3. 다른 여행자에게 공유 (블로그보다 훨씬 쉬움)
4. 공유한 일정에 대한 리워드 획득 가능

## 🎨 사용자 경험 (UX) 철학

### 1. "쉽고 직관적"

- 복잡한 기능을 숨기고, 필요할 때만 표시
- 드래그 앤 드롭 같은 자연스러운 인터랙션
- 모바일에서도 불편함 없는 UI

### 2. "유연하고 자유로운"

- 사용자가 원하는 대로 커스터마이징
- 필수 입력 최소화
- 여행 스타일에 맞게 조정 가능

### 3. "연결되고 통합된"

- 지도-일정-비용이 하나로 연결
- 예약 정보가 일정에 자동 반영
- 모든 정보가 한 곳에

## 🔥 Why Now?

### 1. 시장 타이밍

- 코로나 이후 여행 수요 폭발적 증가
- MZ세대의 개인화된 여행 선호
- 디지털 노마드 증가

### 2. 기술 성숙도

- 구글맵 API 활용 가능
- AI 기술로 자동화 가능
- 클라우드로 저렴하게 서비스 가능

### 3. 경쟁 우위

- 실제 사용자(개발자)의 Pain Point에서 시작
- 기존 서비스들의 기능을 통합
- 빠른 MVP 개발 및 검증 가능
